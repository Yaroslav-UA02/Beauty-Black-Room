ПРОМПТ — BEAUTY BLACK ROOM WEBSITE (OTHERWORLDLY EDITION)

КОНЦЕПЦІЯ
Сайт-вітрина для Beauty Black Room — магазин оригінальної косметики, парфумерії та засобів догляду в Мукачево.
Головний принцип нової версії:
Сайт виглядає як об'єкт із паралельного всесвіту. Не лендинг. Не маркет. Інтерактивний простір, де кожен скрол — це подорож. Натхнення: Marcelo Design X, Studio Freight, Basement Studio.

ТЕХНОЛОГІЇ
React + Vite (один проект)
GSAP + ScrollTrigger — всі scroll-анімації
Three.js або React Three Fiber — 3D фони та ефекти
Framer Motion — мікро-анімації UI
CSS Modules або Tailwind
Google Fonts
Mobile-first / Breakpoints: 768px, 1200px

CURSOR (ОБОВ'ЯЗКОВО)
Кастомний курсор:

Маленьке коло 8px — слідує точно за мишею
Велике коло 40px — з lerp затримкою
При наведенні на кнопки/посилання — magnetic pull ефект (курсор притягується до елементу)
При наведенні на фото — курсор трансформується в "VIEW" або іконку


SCROLL ENGINE
Всі секції побудовані на scroll storytelling (як у Marcelo Design X):
ScrollTrigger.create({
  scrub: true,         // прив'язка до скролу, не таймеру
  pin: true,           // деякі секції "заморожуються" на час анімації
  anticipatePin: 1
})
Принцип: користувач не просто гортає — він керує розповіддю своїм скролом.
